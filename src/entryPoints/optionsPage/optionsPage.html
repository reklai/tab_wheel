Cycling Rules
Alt + WheelCycle tabs using the current cycle mode.
Alt + Left ClickOpen the in-page search launcher.
Alt + Middle ClickJump to the most recently used tab.
Alt + Right ClickClose the current tab; return to the most recently used tab first when available.
Modifier-click caveat Some sites and browsers reserve modifier + left/middle/right click combinations for opening links, saving links, selection, context menus, or system shortcuts. If one combo conflicts on your setup, choose another modifier or require Shift.
Extension constraints Page gestures run on normal web pages. Browser UI and internal pages such as chrome://extensions, about:addons, Chrome Web Store, Mozilla Add-ons, PDF viewers, and some restricted pages may block content scripts; use the toolbar popup controls there.
Page scroll filter Normal page scrolling stays native at 1.0x speed and 100% viewport cap. Non-default page-scroll values filter normal vertical wheel scrolling on supported pages.
